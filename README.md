Python Mini Game Project

A simple menu-driven game application built using Python. Users can choose between two games: Stone-Paper-Scissors and Dice Roll.

🚀 Features

- Stone-Paper-Scissors Game  
- Dice Roll Game  
- Random result generation  
- Menu-driven interface  

🛠️ Tech Used

- Python  
- Random Module  
- Functions & Modular Code  

📁 Project Structure

SimpleGame/
│
├── main.py
├── diceroll.py
├── stonepaperscissor.py

▶️ How to Run

1. Clone the repository  
2. Open terminal in project folder  
3. Run:

python main.py

🧠 How It Works

- The user selects a game from the menu  
- The program uses the random module to generate outcomes  
- Game logic is implemented using functions  
- The program loops until the user exits
